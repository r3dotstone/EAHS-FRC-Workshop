\doxysubsection{Detailed Description}
This class impliments a quadrature based encoder attached to the Motor1 connector. 

The \mbox{\hyperlink{class_f_s_m_encoder1}{F\+S\+M\+Encoder1}} class supports a quadrature based encoder attached to the Motor1 connector. The number of counts can be accessed through the get\+Counts and get\+Counts\+And\+Reset functions.
